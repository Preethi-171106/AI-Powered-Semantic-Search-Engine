# 📚 AI Powered Semantic Search Engine

## 📌 Overview

AI Powered Semantic Search Engine is a Retrieval-Augmented Generation (RAG) application that allows users to upload PDF documents, ask questions in natural language, and receive AI-generated answers based on the document content.

## 🚀 Features

- Upload PDF documents
- Extract text from PDFs
- Automatic text chunking
- Semantic search using Sentence Transformers
- FAISS Vector Database
- AI-generated answers using Google Gemini
- Download AI response as a text file
- User-friendly Streamlit interface

## 🛠️ Tech Stack

- Python
- Streamlit
- PyPDF2
- Sentence Transformers
- FAISS
- Google Gemini API
- NumPy

1. Upload a PDF.

4. Get AI-generated answers.
5. Download the response.
